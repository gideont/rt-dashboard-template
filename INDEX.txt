================================================================================
POLLING MECHANISM
================================================================================

Frontend:
  1. useEffect hook runs once on mount
  2. Fetch /api/metrics immediately
  3. Set up setInterval for polling every 1 second
  4. Display chart and update timer
  5. Cleanup: clear interval on unmount

Result:
  â€¢ Charts update every second with fresh data
  â€¢ Polling stops when user leaves page
  â€¢ No memory leaks
  â€¢ Efficient and clean

Code:
  useEffect(() => {
    const interval = setInterval(fetchMetrics, 1000)
    return () => clearInterval(interval)
  }, [])
